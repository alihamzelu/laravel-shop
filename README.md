# Tech World 🛒

A modern e-commerce platform built with Laravel and Tailwind CSS.

Tech World is an online store designed for selling computer and gaming products. 
The project includes a complete shopping experience with product management, cart system, orders, payments, reviews, wishlist and a powerful admin panel.

## 🚀 Features

### User Features
- User authentication and profile management
- Browse products by categories and brands
- Product search and filtering
- Product details with related products
- Shopping cart system
- Wishlist functionality
- Product reviews and ratings
- Checkout and order management
- Online payment integration (Zarinpal)
- Order history

### Admin Panel Features
- Built with Filament Admin Panel
- Product management
- Category and brand management
- Order management
- User management
- Review management
- Deal and discount management
- Role & permission management using Spatie
- Support ticket management

## 🛠 Technologies

- PHP 8+
- Laravel
- Laravel Blade
- Tailwind CSS
- FilamentPHP
- MySQL
- Spatie Laravel Permission
- Vite
- JavaScript
